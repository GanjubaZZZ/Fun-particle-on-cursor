# Fun-particle-on-cursor 🔥

Невелика програма на Python із використанням Pygame, яка створює ефект трьох вогняних куль, що обертаються навколо курсора миші та випускають частинки вогню.

## Принцип роботи

- Програма відстежує позицію курсора миші в реальному часі.
- Навколо курсора по колу обертаються 3 вогняні кулі з рівномірним зсувом фаз (120° одна від одної).
- Кожна куля щокадру випускає частинки у випадковому напрямку.
- Частинки мають фізику: швидкість, гравітацію (піднімаються вгору і сповільнюються), затухання кольору та прозорості.
- Колір частинок змінюється з часом: жовтий → помаранчевий → червоний → темний, імітуючи згасання іскри.
- Коли частинка повністю згасає — вона видаляється.

## Використані технології

- **Python 3.12**
- **Pygame 2.6+** — графічний рушій для 2D-рендерингу, обробки подій миші та клавіатури
- **math, random** — для обчислення орбіт і випадкового напрямку частинок

## Як використовувати

1. Встанови залежності:
   ```bash
   pip install pygame
   ```
2. Запусти програму:
   ```bash
   python fireballs.py
   ```
3. Рухай мишею — вогняні кулі слідуватимуть за курсором.
4. Вийти — натисни `ESC`.
